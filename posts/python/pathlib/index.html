pathlib 쓰는 법

2022/01/10 10:53:50 Mon

python

pathlib이란

  • 경로를 객체로 다룰 수 있다
  • 공식 문서

경로 다루기

from pathlib import path

>>> from pathlib import Path
>>> p = Path()
>>> p
PosixPath('.')
>>> p = p / "디렉토리"
>>> p
PosixPath('디렉토리')
>>> p /= "파일"
>>> p
PosixPath('디렉토리/파일')
>>> p = p.with_suffix(".txt")
>>> p
PosixPath('디렉토리/파일.txt')

Path() 객체에서 찾기

>>> p.name
'파일.txt'
>>> p.parent
PosixPath('디렉토리')
>>> p.stem
'파일'
